# Data-Bootcamp-Python-Pandas
# 📊 Data Technician Bootcamp – Python Data Analysis Project

## 📖 Overview
This project was completed as part of a **Data Technician Bootcamp (Week 6 – Python)**, focusing on practical data analysis using real-world retail and sales datasets. The work was carried out in **Google Colab notebooks**, where multiple datasets were explored, cleaned, and analysed using Python. 【1-38adb9】  

The project demonstrates core data analysis techniques using **pandas** alongside data visualisation tools to extract insights and support data-driven decision-making.

---

## 🛠️ Key Skills & Techniques

### 🐼 Data Manipulation with pandas
- Filtered and queried datasets to extract meaningful subsets of data  
- Used `.loc[]` and `.iloc[]` for precise row and column selection  
- Applied common data manipulation functions such as:
  - `groupby()` for aggregating sales metrics  
  - `sort_values()` to rank products and performance  
  - `dropna()` and `fillna()` to handle missing values  
- Transformed raw retail data into structured, analysis-ready formats  

---

### 🧹 Data Cleaning & Preparation
- Identified and handled missing or inconsistent data  
- Standardised datasets for accuracy and consistency  
- Prepared transactional and retail datasets for analysis workflows  

---

### 📊 Data Exploration & Analysis
- Analysed retail and sales data to identify:
  - Trends in product performance  
  - Key revenue drivers  
  - Patterns in customer purchasing behaviour  
- Used grouping and aggregation to summarise large datasets  

---

### 📈 Data Visualisation
- Created visual insights using:
  - Built-in pandas plotting  
  - `matplotlib` for customised charts  
- Visualised:
  - Sales trends over time  
  - Product/category performance comparisons  
  - Key summary statistics
